Software Installieren:

Um Software auf diesem System zu installieren gibt es mehrere Möglichkeiten:



Möglichkeit 1: Suchen Sie im Startmenü nach "Software". Dort sollte ein Programm
erscheinen, mit welchem sich ganz leicht beliebige Software installieren lässt.

Möglichkeit 2: Öffnen Sie die Konsole (STRG + ALT +T) und geben Sie
"sudo ep -i [Programmname]" ein.

Weitere Möglichkeiten wären noch die Befehle "sudo apt install ...", "sudo flatpak install ..." und "sudo snap install ..."
